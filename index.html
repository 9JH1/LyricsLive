README

welcome to LyricsLive, this program works with the spotify desktop app to give better lyrics support,

Windows Guide

download the script below

windows download

once that script is running ensure that spotify is on

when all of that is done open this link


if the website shows "undefined" check that the main.exe script is both running and not showing any errors, to tell if there are errors check the text on the main.exe script window, if there is any point where the word "error" is found then try to open the spotify app and selet the "x" instead of the "-" button, for this to work you need to have "minimise on close" on in spotifys preferences.


Linux Guide

download the script below

linux download ( dbus )

ensure that python is installed

ensure that all the modules used in the main_linux.py file are installed either in the same venv as the script or globaly, to install packages globally use --break-system-packages at the end of the pip install command

after all of that is done, and working go to this link.
